# Yousef Kaddoura — Academic Website

## Setup

1. Create a GitHub repo named `YOUR-USERNAME.github.io` and upload the contents of this folder to the root.
2. Add your photo at `images/me_33.JPG`
3. Add your CV at `files/Yousef_Resume_updated.pdf`
4. Open `teaching.html` and replace every `Course Name Here` with your actual course titles.

Your site will be live at `https://YOUR-USERNAME.github.io`.
